Alright, you're looking to contribute? Cool.
Fork the project, pull it.

### Let me lay some basic ground rules:
- Keep files organized and named appropriately.
- The mod *MUST* compile when merged. Save everyone a headache.
- Do not contribute binary blobs, or obfuscated code.
- Use the latest version of Melonloader and Pawperty Damage when building.
- It is expected that you are respectful. Give some baseline respect when making PRs or issues.

### Regarding generative AI use in the project
Generative AI is impressive. We can all say that. Bunch o' math.
However, it becomes a problem when it encroaches on *other people's work*.
I forbid direct use of AI to write code to this repository, that includes agentic AI, *however*:
- Asking questions to generative AI is not encouraged, but allowed. It'd be preferred if you headed to a forum or the Issues tab.
- Using it to validate, a second pair of eyes for simple things like typos, is allowed.

I encourage, if you ask it, to make a thought for yourself *FIRST* and test it.
You'll get further than you might think by doing that.

I, and many others, would value crappy code over pristine code you got from an LLM.
